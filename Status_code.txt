				Все статус коды в программировании:
1xx 	-	Информационные

2xx	-	Успех

3xx	-	Перенаправление

4xx	-	Ошибка клиента (Ошибка Frontend)

5xx	-	Ошибка Сервера (Ошибка Бекенд)





1xx: Информационные
____________________________________________________________________________
Эти коды информируют, что запрос был принят и обрабатывается.
100 Continue — Клиент может продолжить отправку запроса.
101 Switching Protocols — Сервер согласился изменить протокол.
102 Processing — Запрос обрабатывается, но ответ пока не готов.
103 Early Hints — Указывает на предстоящую информацию в заголовках.








2xx: Успех
____________________________________________________________________________
Означает, что запрос успешно выполнен.
200 OK — Успешный запрос.
201 Created — Ресурс успешно создан.
202 Accepted — Запрос принят для обработки, но не завершён.
203 Non-Authoritative Information — Информация может отличаться от исходной.
204 No Content — Нет содержимого, но запрос выполнен.
205 Reset Content — Запрос выполнен, рекомендуется сбросить состояние.
206 Partial Content — Возвращается часть содержимого (например, по Range-запросу).
207 Multi-Status — Множественные результаты (например, для WebDAV).
208 Already Reported — Результат уже сообщён ранее.
226 IM Used — Сервер использовал метод Delta Encoding.







3xx: Перенаправление
____________________________________________________________________________
Указывает, что клиенту нужно выполнить дополнительные действия.
300 Multiple Choices — Несколько возможных ответов.
301 Moved Permanently — Ресурс перемещён на новый URL.
302 Found — Ресурс временно перемещён.
303 See Other — Доступен по другому URL через GET.
304 Not Modified — Ресурс не изменён.
305 Use Proxy — Доступ через прокси (устарел).
306 Switch Proxy — Зарезервирован (не используется).
307 Temporary Redirect — Временное перенаправление.
308 Permanent Redirect — Постоянное перенаправление.





4xx: Ошибка клиента
_________________________________________________________________
Указывает, что проблема в запросе клиента.
400 Bad Request — Некорректный запрос.
401 Unauthorized — Требуется авторизация.
402 Payment Required — Зарезервирован для будущего использования.
403 Forbidden — Доступ запрещён.
404 Not Found — Ресурс не найден.
405 Method Not Allowed — Метод запроса не поддерживается.
406 Not Acceptable — Невозможно предоставить содержимое в приемлемом формате.
407 Proxy Authentication Required — Требуется авторизация прокси.
408 Request Timeout — Тайм-аут запроса.
409 Conflict — Конфликт запроса.
410 Gone — Ресурс удалён.
411 Length Required — Требуется указать длину содержимого.
412 Precondition Failed — Условие в заголовке запроса не выполнено.
413 Payload Too Large — Содержимое слишком велико.
414 URI Too Long — URI слишком длинный.
415 Unsupported Media Type — Неподдерживаемый тип данных.
416 Range Not Satisfiable — Диапазон не может быть предоставлен.
417 Expectation Failed — Ожидание не оправдалось.
418 I'm a teapot — Шутливый код (RFC 2324).
421 Misdirected Request — Запрос направлен на неподходящий сервер.
422 Unprocessable Entity — Необрабатываемая сущность.
423 Locked — Ресурс заблокирован.
424 Failed Dependency — Зависимость не выполнена.
425 Too Early — Рано для обработки запроса.
426 Upgrade Required — Требуется обновление протокола.
428 Precondition Required — Требуется выполнение предварительного условия.
429 Too Many Requests — Слишком много запросов.
431 Request Header Fields Too Large — Заголовки запроса слишком большие.
451 Unavailable For Legal Reasons — Недоступно по юридическим причинам.








5xx: Ошибка сервера
____________________________________________________________________________
Указывает, что сервер не смог выполнить запрос.
500 Internal Server Error — Внутренняя ошибка сервера.
501 Not Implemented — Метод не реализован.
502 Bad Gateway — Ошибка шлюза.
503 Service Unavailable — Сервис недоступен.
504 Gateway Timeout — Тайм-аут шлюза.
505 HTTP Version Not Supported — Версия HTTP не поддерживается.
506 Variant Also Negotiates — Ошибка согласования.
507 Insufficient Storage — Недостаточно памяти.
508 Loop Detected — Обнаружен бесконечный цикл.
510 Not Extended — Требуется расширение.
511 Network Authentication Required — Требуется сетевая аутентификация.
